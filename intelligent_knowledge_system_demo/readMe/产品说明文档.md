# 超智通 - 超来科技智能知识库系统

## 产品概述

**超智通**是一款基于RAG（检索增强生成）技术的企业级智能知识库问答系统。该系统能够将企业文档转化为可智能检索的知识库，通过大语言模型为用户提供精准、高效的问答服务。

---

## 核心功能

### 1. 智能文档管理
- **文档上传**：支持 Word (.docx) 文档上传（PDF 支持开发中）
- **自动分块处理**：采用智能文本分割技术，将长文档切分为适合检索的语义片段
- **向量化存储**：使用先进的Embedding技术将文档内容转化为向量，存储于ChromaDB向量数据库

### 2. 智能问答系统
- **语义检索**：基于向量相似度检索，精准定位相关问题答案
- **上下文增强**：自动整合检索到的相关文档片段，构建丰富的上下文提示
- **大模型生成**：调用通义千问等大语言模型，生成准确、自然的回答
- **检索溯源（基础版）**：展示检索到的原始文档片段（相似度分数、来源文档标识等详细追溯信息开发中）

### 3. Web交互界面
- **简洁美观的UI**：现代化的界面设计，提供流畅的用户体验
- **单轮问答**：支持基于知识库的智能问答（多轮对话开发中）
- **文档管理**：便捷的上传和管理企业文档

## 产品特性

### 1. 高精度语义检索
- 采用先进的Embedding技术，将文本转化为高维向量
- 支持Top-N相似度检索，精准定位相关内容
- 批量处理机制，高效处理大规模文档

### 2. 多模型支持
- 默认集成阿里云通义千问系列模型
- 支持DeepSeek、腾讯混元等多种大模型
- 灵活配置，可根据需求切换不同模型

### 3. 中文优化
- 针对中文文档特别优化
- 支持中文文件名自动转拼音处理
- 使用jieba等中文分词工具

### 4. 数据持久化
- 向量数据本地持久化存储
- 支持多集合管理，便于分类存储不同文档
- SQLite后端，轻量且可靠

## 快速开始

### 环境要求
- Python 3.11+
- 阿里云API Key（用于访问通义千问模型）

### 安装步骤

1. **安装依赖包**
```bash
pip install -r requirements.txt
```

2. **配置API Key**
```bash
# 设置环境变量
set DASHSCOPE_API_KEY=your_api_key_here
```

3. **启动服务**
```bash
python RAG_Project/app.py
```

4. **访问系统**
打开浏览器访问：http://127.0.0.1:5555

### 使用流程

1. **上传文档**
   - 访问 `/document_upload` 页面
   - 选择 Word (.docx) 文档上传
   - 系统自动解析并建立向量索引

2. **开始问答**
   - 访问首页 `/chat` 进入聊天界面
   - 输入问题，系统返回智能回答
   - 查看引用的文档来源

## API接口说明

### 1. 文档上传
```
POST /document_upload/
Content-Type: multipart/form-data

```
参数:
- file: 文档文件 (.docx)

返回: 重定向到上传页面
```

### 2. 智能问答
```
POST /chat/
Content-Type: application/json

请求体:
{
    "message": "用户问题"
}

返回:
{
    "response": "AI回答",
    "search_results": "检索到的相关文档内容"
}
```

## 配置说明

### 模型配置 (models.py)

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| ALI_TONGYI_TURBO_MODEL | qwen3.5-flash | 对话模型 |
| ALI_TONGYI_EMBEDDING_V4 | text-embedding-v4 | 向量化模型 |
| ALI_TONGYI_URL | dashscope.aliyuncs.com | API地址 |

### 系统参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| chunk_size | 300 | 文档分块大小 |
| chunk_overlap | 30 | 分块重叠大小 |
| n_results | 5/10 | 检索返回结果数 |
| temperature | 0 | 模型生成随机性 |
```
